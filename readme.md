# 💱 Currency Converter (Python Tkinter)

A simple **Currency Converter Desktop Application** built using **Python and Tkinter** that converts currencies using live exchange rates from an API.

---

## 🚀 Features
- Convert between multiple international currencies
- Real-time exchange rates using ExchangeRate API
- Simple and user-friendly GUI
- Convert, Clear, and Reset options
- Supports 20+ currencies

---

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
- Requests Library
- ExchangeRate API
